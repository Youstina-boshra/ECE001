programming languages uses
Python it is used by software engineers and back-end Web developers and well-suited for scientific computing
Java Java enables the downloading of applets from websites, which enable browsers to perform additional functions.
HTML it is used extensively in Web development. HTML is the code that serves as the foundation of Web pages, allowing people to create and structure electronic documents for viewing online
C it is used by software developers and systems analysts. Programmers use C to create applications that integrate with operating systems
C++ Programmers use C++ to create games, graphics, and office applications.
PHP Developers use PHP as an open-source language to create dynamic Web pages. Widely used platforms such as WordPress and Drupal work cohesively with PHP.
SQL enables programmers to create, read, update, and delete information in a database. Companies use SQL to gather data.
